Colophon

To generate this website, I'm using some sofware I wrote called “Corpsblog.” Partly, I did this because I can, and I enjoy writing sofware. Another reason is that I was able to set things up for living in Africa with limited internet connectivity. It's designed so I can get the blog all ready for publication with no internet connection, and then when I'm in an internet cafe, type one command and have everything updated all at once. The generated website is just a bunch of files (a “static site”) that I host on Github, along with my other websites.
